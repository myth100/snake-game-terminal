# snakeApp

> This app I made by using [create-ink-app](https://github.com/vadimdemedes/create-ink-app)


## Install

```bash
$ npm install --global snakeApp
```
## To Run
```
$ snakeApp

> Use arrows to turn
```
## CLI

```
$ snakeApp --help

  Usage
    $ snakeApp

  Options
    --name  Eugene

  Examples
    $ snakeApp --name=Eugene
    Hello, Eugene
```
